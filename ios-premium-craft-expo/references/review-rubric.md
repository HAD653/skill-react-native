# Review Rubric

Score each screen from 0 to 100.

## 1) Typography and Readability: 0 to 20

Check:
1. consistent scale usage;
2. clear hierarchy;
3. safe behavior with larger text settings.

## 2) Color, Contrast, Materials: 0 to 20

Check:
1. legibility in light and dark;
2. no low-contrast dark mode traps;
3. translucent surfaces remain readable;
4. fallback strategy for reduced transparency.

## 3) Spacing and Hierarchy: 0 to 15

Check:
1. consistent rhythm;
2. alignment quality;
3. clear visual grouping.

## 4) Components and Affordances: 0 to 15

Check:
1. touch target minimum respected;
2. interaction states explicit;
3. primary action placement is obvious.

## 5) Motion and Feedback: 0 to 15

Check:
1. motion has purpose;
2. motion remains subtle;
3. reduced-motion mode has safe alternatives.

## 6) Performance Feel: 0 to 10

Check:
1. list scrolling and transitions feel smooth;
2. visual effects do not degrade interaction quality.

## 7) Brand Distinctiveness: 0 to 5

Check:
1. visual identity is intentional;
2. interface avoids template-looking patterns.

## Red Flags Cap

If any red flag is present, cap total score at 70 until fixed:
1. raw color and spacing chaos;
2. tiny interactive targets;
3. decorative motion with no reduced-motion fallback;
4. generic kit appearance with no clear visual voice.

## Severity Mapping

1. 0 to 59: major redesign required.
2. 60 to 74: production risk; fix CRITICAL and HIGH issues.
3. 75 to 89: solid quality with targeted polish needed.
4. 90 to 100: premium production quality.
